# Numpy implementation of basic PyTorch modules
Modules included:
1. Modules for MNIST dataset training and evaluation
    1. SGD
    2. SGDMomentum
    3. ReLU
    4. Softmax
    5. Dense
    6. CrossEntropy
2. Modules for CIFAR dataset training and evaluation
    1. convolve
    2. Conv2D
    3. Pooling2D
    4. BatchNorm
    5. Flatten
    6. Dropout
